- [ ] 画面为SCN002殿内纯空场；没有人物、群像、人体局部、坐席、巨树或树下讲经区。
- [ ] 殿内占据主体，门外只提供方向与纵深；没有变成外廊、广场、庭院或SCN003。
- [ ] 只有画框边缘出现少量巨柱；中央与右侧地面连续、宽阔、无遮挡，能容纳群像与摄影机进入。
- [ ] 巨柱尺度统一，柱顶多数在画外；没有近景巨柱、远景普通细柱互相冲突。
- [ ] 屋顶极高，只在顶缘出现少量巨型梁架；没有低矮压迫屋盖、密集小梁或密集斗拱。

- [ ] 门外不是纯白矩形或神圣入口；门内外亮度差克制，灰度与地面连续。
- [ ] 没有顶光、天窗、屋顶开口、垂直光柱、丁达尔光、聚光或金光。
- [ ] 建筑具有中国传统柱梁檐门关系，但不形成现实寺院、具体朝代复原、日式禅院或现代建筑。
