## 5. 逐图验收清单

### 5.1 硬性项｜任一不通过即退回

- [ ] 画面为门外略偏轴位置看向殿内，不是正中建筑立面展示。
- [ ] 门外石台只占画面下缘小部分，殿内与人物占据画面主体。
- [ ] 入口和殿内中央无柱；巨柱只出现在画框边缘和左右墙边。
- [ ] 屋顶高，主梁少；没有低矮屋盖、密集梁架或密集斗拱。
- [ ] 二十余名女性修行者完整留发并梳低位圆髻；没有女性剃发、男性面孔或男性身体。
- [ ] 群像年龄、脸型、体型、发际线、服装灰度和褶皱存在自然差异；没有复制人群和军队式排列。
- [ ] 年轻女性位于后层略偏左，不居中、不独坐、不穿特殊服装、不接受独立聚光。
- [ ] 多数人闭眼或垂目；年轻女性身体与头部不动，双眼睁开并看向摄影机。
- [ ] 焦点落在年轻女性眼睛和面部，前后群像仍可辨认，且没有人物遮挡她的头部与视线。
- [ ] 光线从摄影机后方宽柔进入，殿内保持中灰和深灰细节；没有纯黑洞口、顶光、神光或金光。
- [ ] 地面、木柱、服装干燥哑光；没有湿亮反射、水墨滤镜、文字、水印或拼图。

### 5.2 软性项｜偏差必须修订复检

- [ ] 巨型木构尺度清楚，但建筑没有压过人物面部和群像关系。
- [ ] 群像自然错落，年轻女性第一眼仍属于群体，第二眼可通过焦点找到。
- [ ] 殿内中央地面保持完整，人物与柱子没有切成多个狭窄区域。
- [ ] 服装保持中国式交领、宽袍和披衣关系，不像现代制服或现实宗派僧服。
- [ ] 黑白灰、木石材质与`水墨佛国.png`属于同一视觉世界。
