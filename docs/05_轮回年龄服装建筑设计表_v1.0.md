## 1. 总体规则

1. 佛国女主固定20—22岁；镜头29回到佛前时恢复同年龄外观。
2. 轮回状态继承同一组面部和手部身份锚点，但年龄按对应历史时刻独立变化。
3. 画面与资产名称不出现真实历史人物姓名、朝代政权名称或可辨识真实文字。
4. 朝代信息只作为内部服装、建筑、发式、材质和道具设计依据。
5. 可确定事件年龄时使用具体年龄；生年不详时使用生产年龄范围，并注明置信度。
6. 服装与建筑必须服务2—4秒的情绪识别，不堆砌考据细节，不做艳俗古装混剪。

## 4. 念珠锁定

- 正式生产资产ID：`ASR-PRP-P0-001-念珠`。
- 旧ID映射：`ASR-PRP-P0-001-灰白念珠 → ASR-PRP-P0-001-念珠`。
- 单珠直径：10毫米。
- 可接受材质方案：灰白哑光石质/菩提质感；深棕檀木。
- 传统基数：108颗，仅作为内部道具常识登记。
- Prompt规则：不写“108颗”，不要求画面完整展示或可数清全部珠数；描述为“完整连续的一串修行念珠，珠径统一，长度足以完成稳定拨珠、停住、握持和逆拨动作”。
- 连续性锚点：10毫米珠径、材质方案、线体颜色与柔度、珠孔、表面磨损、手部接触点和逆拨方向。

## 5. 其他锁定项

- 青绿落叶：窄椭圆真实叶，冷灰青绿、真实叶脉、轻微湿润感，不对应具体神树。
- 佛前归来：复用最初佛国庭院骨架，不新建独立空间。
- 片名卡与落叶影转场：阶段2制作3个候选方案供确认。
- 诏书和印面：使用不可辨虚构笔画与印面，不出现真实姓名、朝代或政权。
- 女主身份：阶段2先制作身份候选，再确认最终脸型、眉骨、眼型、唇形和手部锚点。
- 女主声音：阶段2先交付音色资产Prompt，由用户选择平台生成；佛声和众声只用镜头Prompt控制。
