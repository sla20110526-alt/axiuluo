# 《阿修罗》阶段1｜资产生产计划

版本：v1.0（阶段1正式锁定版）  
分析范围：镜头01—30，共214秒  
资产记录：102条，其中88项生产资产、14项关系与连续性记录

## 1. 总体优先级

- P0：41条记录。必须先锁定女主身份、佛非正面存在、佛国主场景、核心光影、念珠、青绿落叶、关键VFX、核心声音与连续性关系。
- P1：54条记录。随P0配套制作轮回状态、重点场景、关系人物、叙事道具、群演配置和轮回声音。
- P2：7条记录。短促快闪或低频背景资产，在主身份与主空间通过审批后后置生产。

## 2. 生产顺序

1. 风格与真实度基准测试：女主身份、真实皮肤、低饱和摄影、青绿落叶色彩上限。
2. 女主身份、佛国基础服装、被照见状态、佛前归来状态和女主声音需求。
3. 佛非正面存在、比丘尼群像与佛国主空间。
4. 殿内/庭院空间视图、主场景光影、念珠和落叶形制。
5. 主观目光、失焦、落叶下沉、心远层退和动作匹配切技术测试。
6. 佛国声音秩序与中断母题。
7. 女主多世轮回服装/状态、轮回关系人物与人间场景。
8. 发簪、诏书、印章、酒杯、白绫等叙事道具及轮回声音。
9. 片名卡、落叶影转场、死亡叠闪和佛前归来总装验证。
10. P2快闪背景与补充资产。

- 可复用：女主基础身份、佛非正面局部、佛国殿内/庭院空间骨架、念珠、青绿落叶、佛国声音秩序、呼吸/心跳母题、群演配置规则。

## 5. 需要技术测试的资产

1. 女主跨14状态身份一致性。
2. 真实手部与念珠逆拨、握簪、推杯、落印、接叶的连续接触。
3. 主观真实眼睛无魔眼/暧昧误读。
4. 群体逐步失焦但人物轮廓不崩坏。
5. 青绿落叶慢速下沉与叶形稳定。
6. 心远空间分层后退而女主尺寸不变。
7. 九段轮回动作匹配切和屏幕方向统一。
8. 镜头29极轻口型；不自然时切换画外音。
9. 原生声画联合生成中的梵呗、木鱼、念珠、呼吸与中断可控性。
