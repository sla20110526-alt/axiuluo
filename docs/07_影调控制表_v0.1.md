## 1. 项目总影调

| 控制项 | 全片统一规则 |
|---|---|
| 明度 | 中低明度；佛国保留雾中层次，轮回保持材质可读，黑场才允许接近纯黑 |
| 对比 | 总体中低对比；心理转折通过局部对比与焦点变化加强，不用全局硬对比 |
| 饱和度 | 总体低饱和；青绿落叶为全片相对彩度上限，轮回时代主色低于其视觉权重 |
| 色温 | 佛国冷中性；实景烛火保留局部暖色；轮回按材料与光源轻微变化，不用一镜一滤镜 |
| 黑位 | 冷感、稳定、略厚但不死黑；不同轮回共用同一黑位逻辑 |
| 高光 | 柔和滚降，保留皮肤、雾、织物、珠体和叶片细节；禁止大面积纯白过曝 |
| 肤色 | 真实、克制、跨轮回稳定；允许年龄与环境造成微小变化，不允许随主色明显染色 |
| 材质 | 石木织物低反光；漆面、金属和烛火只在物理合理位置形成高光 |
| 空气感 | 佛国湿冷薄雾；轮回按深院湿气、室内烛烟、宫廷暗空气、驿站尘土等变化 |
| 背景清洁度 | 构图清楚但不人工洁净；无连续性价值的普通细节并入场景，不堆道具 |
| 禁用影调 | HDR提亮、黑金大片、青橙商业片、霓虹赛博、糖水古装、仙侠金光、全画面复古黄罩 |

## 2. 色彩权重与统一标尺

### 2.1 画面占比

- 母体中性色：70%—80%。
- 当前轮回命运主色：15%—25%。
- 强调色：0%—5%。
- 青绿落叶出现时，其局部色彩辨识度必须高于同镜其他色彩。

### 2.2 相对彩度

- 将最终确认的青绿落叶标准资产设为相对彩度标尺100。
- 佛国普通材料色建议处于20—35区间。
- 轮回命运主色建议不超过60—70区间。
- 掌纹暗血色建议不超过50，且只出现极小面积和极短时间。
- 数值为内部相对控制，不是直接写入Prompt的色彩值；正式色度在测试图回流后统一测量。

### 2.3 跨镜统一

- 女主皮肤、眼白、黑发、瞳孔和唇色建立单独身份基准，不纳入每世综合色罩。
- 所有轮回共用同一黑位、高光滚降、颗粒尺度和皮肤锐度。
- 色彩变化优先来自服装、墙体、帘幕、漆木、尘土和实景光源。
- 剪辑衔接同时匹配动作方向、主体明度和轮廓面积，避免只靠换色制造转场。

## 7. 调色与回流验证门

P0测试资产回流后，按以下顺序验证：

1. 先锁定女主肤色、黑位、高光和皮肤纹理。
2. 再锁定青绿落叶标准色及其在佛国灰阶中的辨识度。
3. 用SCN002母图验证佛国石木灰与殿门光柱。
4. 用一组轮回缩略色稿验证九段色彩弧线，不生成正式轮回资产。
5. 检查各命运主色是否来自材料，而非全画面色罩。
6. 检查镜头27是否保持尘土、暑热与军队压力，镜头29是否完全回归佛国母体。
7. 主创确认后再记录正式色值、显示环境和调色参考图。
