# 《阿修罗》阶段2A｜禁止改写项清单

版本：v1.0（主创锁定版）  
建立日期：2026-07-13  
确认日期：2026-07-13  
规则系统：v1.0.1  
模型适配卡：MA-SD2.0-BASE-v1.0  
适用范围：阶段2资产Prompt、外部生成回流、阶段3分镜与正式镜头Prompt、后续修订  
当前状态：主创已确认锁定；零例外

## 1. 执行效力

1. 本清单是《阿修罗》项目跨阶段的硬性边界，优先于单条资产Prompt、并行分工包和外部平台默认审美。
2. 任何Prompt、回流资产、分镜或镜头方案触犯本清单，均判定为硬性不合格，不得继续继承或标记为可调用。
3. 并行执行单元、外部生成平台和后期修订不得以“效果更好”“模型更稳定”或“后期可以修”为由绕过本清单。
4. 当前无例外。任何变更必须先提交影响说明并获得主创书面确认。

## 2. 项目核心锁定项

### 2.1 技术与成片形态

1. 全片锁定为21:9、4K、24fps、纯真人写实电影质感。
2. 有效画面为3840×1648；如需3840×2160兼容版，使用上下黑边，不改变有效构图。
3. 禁止动画、二次元、游戏CG、虚拟偶像、仙侠海报、玄幻短剧或水墨特效质感。

### 2.2 女主身份与年龄

1. 女主佛国视觉年龄锁定为20—22岁；镜头29恢复同一年龄外观。
2. 所有轮回保持同一演员身份，年龄按各事件时刻变化，不以换脸表达时代变化。
3. 最终脸型、眉骨、眼型、唇形、面部比例、基本发际关系、手掌比例和主要指节特征，必须在阶段2 P0身份候选中确认。
4. 女主身份通过前，不得扩展轮回服装和状态资产。
5. 禁止网红脸、塑料皮肤、蜡像脸、艳俗古装脸、邪魅妆、发光眼、魔纹、大表情和用宽袖遮挡手部结构问题。

### 2.3 佛与阿修罗的展示边界

1. 佛只允许通过袈裟边缘、肩线、垂落手指、局部背影或侧后轮廓建立存在。
2. 禁止佛正面身份照、正面神像化、转头选择女主、私人凝视、金身、神迹或情感暧昧表演。
3. 不实体化阿修罗，不建立怪物资产；禁止魔纹、发光眼、能量爆发、空间爆裂和怪物化形象。

### 2.4 佛国与轮回世界

1. 佛国服装、建筑和地域不绑定现实朝代、地点、寺院或宗派，是意向上的天国。
2. 轮回资产匿名登记，只记录年龄、身份、服装、建筑、事件语汇和置信度，不写真实人物姓名。
3. 轮回历史语汇只服务美术与材料依据，不得生成真实历史人物肖像映射、真实政权名称或跨朝代混剪。
4. 不得使用明清建筑、宫门或服饰语汇覆盖其他时代设计。

## 3. 色彩、光影与材质锁定项

1. 主风格保持“冷寂因果现实主义＋时间考古＋生理心理电影＋传统色历史残响”。
2. 轮回画面统一使用70%—80%母体中性色、15%—25%命运主色、0%—5%强调色的控制逻辑。
3. 青绿落叶为全片唯一明确生命色；其他时代主色的视觉彩度不得超过其约60%—70%。
4. 命运主色必须附着于织物、漆木、石土、尘灰、金属或实景光源，不得使用九套独立全画面滤镜。
5. 禁止金黄圣光、荧光绿、霓虹紫、宝石蓝天空、艳红法衣、青橙商业片、黑金大片、糖水古装和全画面复古黄罩。
6. 女主肤色不得随轮回命运主色漂移。
7. 主光必须来自天空、山雾、殿门、庭院、真实烛火或时代空间中的可见光源。
8. 人物、道具、地面和背景必须共享同一光源方向、色温、阴影和空气介质。
9. 禁止人物自动补光、正面美容柔光、无来源轮廓光、HDR提亮暗部、金色圣光和舞台追光。
10. 禁止塑料、树脂、过度抛光、游戏贴图、统一做旧和无理由潮湿；皮肤、织物、石木、漆面、金属、纸绢和叶片必须保持材料差异。

## 4. 场景与视图锁定项

1. 场景只执行阶段1确认的SV0或SV1-lite，不制作SV2或标准五视图。
2. 阶段3如发现专项机位缺口，必须返回阶段2登记SV3稳定场景视图ID，不得在正式镜头Prompt中臆造稳定视图。
3. 殿内—殿门—庭院—树下讲经区共享唯一客观轴线，不得重置空间关系。
4. `ASR-SCN-P0-002-殿内修行空间`执行`MASTER → AXIS-DOOR / HERO-POS`继承链。
5. `ASR-SCN-P0-003-殿门庭院树下讲经一体场景`执行`MASTER → AXIS-TREE / BRANCH`继承链。
6. 佛前归来复用最初庭院骨架，不生成独立场景母图。
7. 左右方向视图不得使用镜像替代。

## 5. 道具、文字与VFX锁定项

1. 念珠单珠锁定为10毫米；可使用灰白哑光石质／菩提质感或深棕檀木；Prompt不得写精确珠数。
2. 青绿落叶锁定为窄椭圆真实叶、冷灰青绿、真实叶脉、轻微湿润、不发光。
3. 青绿落叶在枝端、下沉、片名影子和掌心接触中保持同一叶形与完好状态。
4. 诏书、印章和其他背景文字使用不可辨虚构笔画与印面，不出现真实姓名、朝代或政权。
5. 全片唯一允许清晰可读的画内文字是《阿修罗》片名。
6. 片名卡与落叶影转场必须在阶段2制作3个候选；片名使用近黑深灰底、灰白枯笔字，不描金、不发光、不加副标题或英文。
7. VFX必须描述可见物理结果；禁止水墨粒子、宣纸叠层、宇宙眼、星河瞳孔、能量波、粒子消散和空间爆炸。
8. 群体失焦不得让人物消失、融化或粒子化；心远层退不得让女主坠落、缩小或漂浮；落叶下沉不得发光或炫技旋转。

## 6. 声音锁定项

1. 全片无非叙事配乐；声音承担节奏、空间和心理转折功能。
2. 女主音色为独立声音资产，由阶段2提供Prompt，用户在外部平台生成并回流确认。
3. 女主基础声纹只锁定稳定音色，不固化具体剧情情绪。
4. 佛声和众声优先由正式镜头Prompt控制，不单独建立音频文件资产；失败时允许后期替换。
5. 镜头22弦乐是场内同期琴声，不是背景配乐。
6. 镜头23隔门喜乐声与杯沿轻碰写入镜头声音Prompt，不新增声音资产。
7. 禁止持续低频铺底、持续哭腔、播音腔、甜美少女音、妖艳反派音和无来源电子音。
8. 念珠、木鱼、梵呗、远磬、呼吸、水滴、心跳、衣料、断弦、印章、酒杯、门轴、风雪和喘息均必须保持可追溯物理来源。

## 7. 镜头级禁止改写项

1. 镜头09继承`ASR-SCN-P0-002-殿内修行空间`和殿内光柱关系，不新增场景或独立光源。
2. 镜头23的隔门喜乐声与杯沿轻碰属于镜头声音Prompt，不新增生产资产ID。
3. 镜头26诏书与毒酒按匿名历史切片和“戏剧压缩”执行，不写真实姓名、朝代或政权。
4. 镜头27以`ASR-SCN-P1-009-盛唐驿站终局空间`为主，使用尘土、暑热和军队压力；风雪只属于跨世反噬叠化，不得成为主环境或服装依据。
5. 镜头29复用`ASR-SCN-P0-003`庭院骨架，只调整听众剪影、人物占位和光影状态。
6. 镜头29短句只允许极轻口型；不自然时改为画外音，不强制可见口型。

## 8. 生产数据与阶段边界锁定项

1. 不得重新命名、拆分、合并、删除或替换102个稳定生产资产ID。
2. 102个稳定ID包括88个视觉／声音生产资产和14个REL关系记录；REL关系不得被普通描述替代或删除。
3. 阶段2不得重新拆解镜头01—30，不得推翻阶段1资产口径；信息不足只登记补充拆解项。
4. 一条资产Prompt只服务一个生产资产ID，或同一资产的一组规定视图／声音样本。
5. 2A未完成并通过P0验证前，不得启动2B批量资产Prompt生产。
6. 女主身份、场景母图或关键P0资产未通过时，不得继续制作其依赖资产。
7. 未经主创确认的视觉或声音回流资产不得标记为可调用，不得进入阶段3正式Prompt。

## 9. 不属于“改写”的待锁定内容

以下内容尚待P0回流或既定审批流程补齐；在本清单边界内完成确认，不视为对本清单的例外：

1. 在既定色彩弧线和权重内建立项目专属HEX、RGB和LUT。
2. 从P0身份候选中选择女主最终脸型、眉骨、眼型、唇形和手部锚点。
3. 在既定方向内校准材质旧化、空气感、饱和度、声音距离、气声比例和母题节奏。
4. 从规定的3个片名卡与落叶影转场候选中选择最终方案。
5. 为已确认资产登记最终镜头使用名、声音使用名和平台引用名。

上述内容一旦经主创确认并写入正式映射或风格文件，即转为新的锁定口径。
