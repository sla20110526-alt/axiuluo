版本：v0.1（主创确认稿）  
建立日期：2026-07-13

模型适配卡：MA-SD2.0-BASE-v1.0  
正式母版：《阿修罗》艺术序章_导演整合修订版（2026-07-12）  
适用范围：镜头01—30及后续《阿修罗》轮回长篇项目的视觉母体  
当前状态：已完成风格设定草案；待主创确认后锁定，不据此启动2B批量资产Prompt

## 1. 项目风格一句话

一部发生在“不存在的时代”中的冷寂宗教现实主义电影：以真实皮肤、石木织物、自然光和生理性微表演呈现“我”的诞生；以低饱和、材料真实且各具传统中国色命运的历史残片呈现千年执取；一切超常只发生在知觉、距离、焦点、动作和声音之中。

## 2. 风格总定义

### 2.1 主体风格：冷寂因果现实主义

- 佛国不是仙境，而是具有湿度、温度、尘粒、重量和声场纵深的真实修行空间。
- 佛不制造神迹，天地不替人物表达情绪；因果通过呼吸错拍、念珠停顿、焦点变化、空间层退和声音中断显形。
- 情绪强度来自人物主动选择及其后果，不来自哭喊、血腥、艳丽色彩或视觉奇观。
- 镜头相信观众能够阅读睫毛、瞳孔、唇线、掌纹和手指压力，不用解释性表演替代心理过程。

### 2.2 材质子系统：时间考古

- 画面中的颜色首先来自材料：石、木、漆、织物、尘土、金属、纸绢、叶片和真实火焰。
- 佛国材质偏湿冷、低反光、轻度风化；轮回材质随时空改变，但保持相同的真实尺度与表面逻辑。
- 纹理只服务触感和时代辨识，不堆积“古意”装饰；不把所有表面统一做旧或做脏。
- 人物皮肤与手部始终是最重要的真实度标尺，不允许场景精致而人物呈现塑料或蜡像质感。

### 2.3 主观子系统：生理心理电影

- 主观异常不改变世界物理，只改变女主的知觉组织。
- 允许变化：焦点、可见空间宽度、声音距离、呼吸近远、空气透视、人物与背景的清晰度层级。
- 禁止变化：凭空神光、眼睛发光、宇宙纹理、魔纹、能量波、空间爆裂、人物无来源漂浮。
- `ASR-VFX-P0-001-主观生理真实目光`必须与客观世界镜头成对存在：主观上仿佛只照见她，客观上佛从未转头或选择任何人。

### 2.4 轮回子系统：传统色历史残响

- 每一轮回拥有一个“命运主色”，但不拥有独立的全画面滤镜。
- 色彩建立于相应时空可成立的染织、漆木、矿物颜料、土石和金属材料之上，再进行心理化编排。
- 所有轮回共用同一女主肤色基准、冷感黑位、高光滚降、低饱和上限和真实材质尺度。
- 轮回色彩形成明确弧线：浅冷蓝 → 短暂暖黄 → 冷青与褪红 → 绛红 → 石青与旧金 → 玄黑与暗朱 → 土褐 → 缟白铁灰 → 佛国冷灰与青绿归来。
- 传统色名用于定义色相倾向与材料来源，不宣称为跨时代固定HEX值；最终色值在P0测试资产回流后按同一显示与调色环境锁定。

## 3. 叙事气质

| 层面 | 执行定义 |
|---|---|
| 精神气质 | 冷、静、准；不故作玄奥，不把宗教感等同于华丽和神秘 |
| 情绪路径 | 秩序 → 被照见的安宁 → 唯一感 → 失去 → 主动执取 → 因果显形 → 千年加速 → 佛前凝固 |
| 观看关系 | 摄影机不审判女主，也不替佛审判女主；镜头观察选择如何发生 |
| 时间质感 | 佛国时间连续、缓慢、可呼吸；轮回时间被压成真实历史残片；镜头29突然恢复静止 |
| 暴力策略 | 不依赖伤口、酷刑和受害者特写；以动作完成、门关闭、道具撞击和后果空间表达 |
| 宗教策略 | 佛以空间、衣料局部、肩线、垂手、听众朝向与声音距离建立存在，不正面神像化 |

## 4. 写实程度与AI真人感控制

### 4.1 写实等级

- 纯真人电影写实，不采用插画、水墨特效、二次元、游戏CG、虚拟偶像或舞台摄影质感。
- “淡墨”只作为雾、低对比、灰阶层次和留白的摄影结果，不表现为墨迹晕染或宣纸纹理。
- 超常现象必须能拆解为摄影、景深、分层合成、速度调整、声音设计或轻量VFX的可见结果。

### 4.2 人物真实度

- 保留毛孔、细小纹理、皮肤薄厚差、自然细纹、微血色和轻微左右不对称。
- 不磨皮，不统一高光，不用大面积柔焦掩盖生成痕迹。
- 眼球保持真实湿润度和自然巩膜颜色，不高亮、不玻璃化、不更换虹膜图案。
- 牙齿不主动展示；嘴唇保持真实唇纹和克制含水度。
- 手部必须有正确掌骨、指节、指甲、掌纹、受力和道具接触，不以宽袖遮挡结构问题。

### 4.3 同一演员控制

- 全部CST状态继承`ASR-CHR-P0-001-女主基础身份`。
- 不可变锚点：脸型、眉骨、眼型、唇形、面部比例、发际基本关系、手掌比例和主要指节特征。
- 可变项：事件年龄、发式、妆容、服装、皮肤生活状态、精神状态和姿态。
- 佛国与镜头29恢复20—22岁外观；轮回年龄按阶段1匿名设计表执行。

## 5. 总色彩系统

### 5.1 母体色

| 色彩角色 | 方向 | 用途 |
|---|---|---|
| 主色 | 近黑、炭灰、石灰、旧木灰 | 建立全片冷寂秩序和跨时空统一性 |
| 辅色 | 真实肤色、纸绢灰白、低饱和土褐、漆木黑 | 保留人物生命与材料差异 |
| 唯一明确生命色 | 冷灰青绿 | 只属于`ASR-PRP-P0-003-青绿落叶`及其连续影像 |
| 轮回命运色 | 浅靛、缃黄、缥青、绛、石青、暗朱、尘赭、缟白、铁灰等低饱和色域 | 区分时空与心理阶段，不改变全片母体影调 |

### 5.2 色彩权重规则

- 每一轮回画面建议由70%—80%母体中性色、15%—25%命运主色、0%—5%强调色构成。
- 青绿落叶是全片色彩辨识度和相对彩度上限；其他时代主色的视觉彩度不得超过落叶约60%—70%。
- 轮回主色必须附着于可见材料或实景光源，不使用无来源的全画面综合色罩。
- 女主肤色不随轮回主色大幅漂移；身份识别优先于时代调色。
- 暗血色只允许作为掌纹中极短暂、极低饱和的一闪，不与青绿落叶竞争。

### 5.3 传统色研究口径

- 采用“历史文献＋文物实物＋染料/颜料材料＋制作工艺”的多重证据方法。
- `缃`按浅黄色域理解，`缥`按青白色域理解，`绛`按深红色域理解；具体数值不跨朝代固化。
- `月白倾向`只作为浅靛低浓度的电影视觉名，不声称其为所有时代相同的实物色。
- `石青倾向`既可来自矿物颜料视觉，也可作为低明度蓝绿色的美术方向；具体载体必须符合当前场景材料。
- 任何色名进入正式资产Prompt前，都要转换为可见色相、明度、彩度、材质载体和光照结果，不能只写文学性色名。

## 6. 光影系统

- 所有主光必须有实景来源：天空、山雾、殿门、庭院环境光、真实烛火或时代空间中的可见光源。
- 佛国光线不主动选择人物；女主处于光柱边缘，而非追光中心。
- 人物、道具、地面和背景共享同一套光源方向、色温、阴影软硬和空气介质。
- 暗部保留层次与真实颜色，不使用HDR式整体提亮。
- 高光应柔和滚降，保留皮肤、织物和珠体细节；不得大面积纯白过曝。
- 禁止正面美容柔光、无来源轮廓光、金色圣光、舞台烟雾光束和人物自动补光。

## 7. 摄影语言

- 画幅固定为21:9，构图重视横向空间中的秩序、距离、门槛和人物边缘位置。
- 佛国段以固定、极缓移动、可呼吸长镜头为主；轮回段以动作方向、手部形状和声音撞击建立切点。
- 特写必须保持生理尺度，尤其是眼睛、嘴唇、掌纹、念珠和叶片；不使用广告式锐化和超微距奇观化。
- 主观镜头可收窄空间、压缩信息和改变焦点层级，客观镜头必须恢复同一空间轴线。
- 镜头焦段、正式机位和运动参数留给阶段3分镜确定；阶段2只锁定摄影气质和空间继承规则。

## 8. 场景体系

### 8.1 佛国

- 意向上的天国，不绑定现实朝代、地域、寺院或具体宗派建筑。
- 使用克制木石结构、真实山雾、低反光表面和自然磨损；不使用漂浮仙山、金殿、祥云、发光经文。
- 殿内—殿门—庭院—树下讲经区共享唯一客观轴线。
- 场景只执行阶段1确认的SV0/SV1-lite；不制作SV2或标准五视图。

### 8.2 轮回

- 每个时空只选择一组最能支持动作与权力关系的建筑语汇，不堆砌时代符号。
- 建筑差异通过尺度、柱网、门窗、案台、帘幕、庭院比例、地面与材料体现，不依赖可读匾额或文字。
- 轮回资产匿名登记；历史依据只服务内部美术、服装和材料设计。
- `ASR-SCN-P1-009-盛唐驿站终局空间`以尘土、暑热、驿站和军队压力为主；风雪仅属跨世反噬叠化。
- 镜头29复用`ASR-SCN-P0-003`庭院骨架，只改变听众剪影、人物占位和光影状态。

## 9. 人物、服装与群像

- 女主身份是全片最高稳定资产；任何时代造型不得通过换脸、夸张妆容或妖异特征表达变化。
- 佛国服装为非朝代、非地域的天国修行服，强调内外层、盘坐重力、袖口功能和低反光天然纤维。
- 轮回服装以事件年龄、身份、动作和建筑语汇为设计依据，传统色必须落在真实织物与工艺逻辑上。
- 比丘尼、听众、群臣与人群保持真实个体差异，禁止同脸复制、同一衣褶复制和整齐到非人化。
- 佛只生成肩背、袈裟边缘、垂落手指、局部背影或侧后轮廓，不生成正面身份照。

## 10. 道具与文字

- 念珠：10毫米单珠；灰白哑光石质/菩提质感或深棕檀木；Prompt不写精确珠数。
- 青绿落叶：窄椭圆真实叶、冷灰青绿、真实叶脉、轻微湿润、不发光。
- 发簪、酒杯、印章、诏书等以重量、接触、动作可读性和连续性优先，不以华丽程度抢夺叙事。
- 诏书与印面使用不可辨虚构笔画，不出现真实姓名、朝代或政权。
- 唯一允许清晰可读的画内文字是《阿修罗》片名。

## 11. VFX与特殊现象

- VFX必须描述可见物理结果，不使用抽象法术词汇。
- 落叶下沉只改变高度与速度，不炫技旋转、不发光。
- 群体失焦保持人物轮廓稳定，不能让群像消失、融化或变成粒子。
- 心远层退保持女主尺寸和坐姿不变，殿、庭院、树与远山逐层远离并褪色。
- 轮回动作匹配切以手部方向、道具接触、轮廓形状和声音撞击为统一依据。

## 12. 声音与视觉关系

- 无非叙事配乐；声音组织承担节奏、空间和心理转折功能。
- 佛国秩序由念珠、木鱼、梵呗、远磬、风声和呼吸建立。
- 异常由少一拍、中断、水滴纵深和距离变化建立，不使用持续低频铺底。
- 轮回由呼吸、心跳、衣料、断弦、印章、酒杯、门轴、风雪和喘息加速。
- 镜头22弦乐是场内同期琴声，不是配乐。
- 女主声纹作为独立声音资产；佛声和众声优先由镜头Prompt控制。

## 13. 资产生成统一规则

- 所有资产必须继承本风格设定、对应生产资产ID、阶段1资产卡和关联REL记录。
- 人物、服装、道具采用中性低反差结构光；场景母图可使用完整项目光影。
- Prompt必须把传统色名翻译为可见色相、明度、彩度、材质和受光结果。
- 资产图保持静态、结构清楚、背景克制，不预演正式剧情镜头。
- 信息不足时登记补充拆解项，不以Prompt擅自补写世界观或历史事实。

## 14. 禁止项总表

- 动画、二次元、游戏CG、仙侠、玄幻海报、短剧磨皮、虚拟偶像质感。
- 塑料皮肤、蜡像脸、网红脸、艳俗古装脸、夸张邪魅妆、发光眼、魔纹。
- 金光佛国、佛正脸、私人凝视、佛转头选择女主、五百罗汉逐一正面实体化。
- 水墨粒子、宣纸叠层、宇宙眼、星河瞳孔、能量爆发、神迹光柱。
- 全画面高饱和传统色滤镜、九段轮回九套互不相关的LUT、肤色随时代主色漂移。
- 明清建筑和服饰语汇跨时代套用、可读真实诏书、真实政权印面和历史人物肖像映射。
- 过度血腥、酷刑展示、真实落泪、持续哭腔、持续低频和非叙事配乐。

## 16. 研究依据说明

传统色部分参考国家颜色体系、中国丝绸博物馆传统染料与色谱重建研究、故宫博物院文物色彩与材料研究。研究结论只用于建立历史材料可信度与电影色域，不将现代色名标准冒充具体朝代的唯一实物色。
