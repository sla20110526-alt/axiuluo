## 02 分析范围与拆解口径

本分析覆盖佛国修行秩序、被照见与主动执取、一念因缘、片名门槛、九段轮回高潮和片尾黑场。拆解遵循以下口径：

1. 需要保持身份、空间、材质、形制、声纹或动作连续性的对象建立稳定生产资产ID。
2. 会被 Seedance 2.0 作为图像、音频或视频参考输入的元素独立登记。
3. 普通砖石、尘土、不可辨背景器物不独立建卡，进入所属场景的材质与陈设规则。
4. 女主跨千年只建立一个基础身份资产；年龄、发式、妆容、服装和精神状态作为 CST 变体。
5. 佛只登记“非正面存在”资产，不建立可见正脸，不产生私人情感表演。
6. 场景级主光、暗区、空气介质和人物活动区受光作为 LGT 资产；阶段3不得临时改写主光。
7. 服装只登记结构、材质和剧本/身份色彩约束；最终色盘留给阶段2A。
8. 声音按可复用母题拆分；不把每一次落珠或呼吸录音拆成新的资产。

## 03 叙事资产主轴

全片资产不是并列清单，而是围绕四条主轴组织：

- 身份轴：`女主基础身份 → 佛国基础状态 → 被照见状态 → 多世轮回状态 → 佛前归来`。
- 因果轴：`念珠秩序 → 停住 → 主动握住 → 逆拨 → 千年动作接力 → 合拢青绿落叶`。
- 空间轴：`殿内偏后 → 殿门 → 庭院树下 → 主观空间变窄 → 客观轴线恢复 → 心远层退`。
- 声音轴：`规律循环 → 少一拍/骤停 → 呼吸与心跳凸显 → 人间冲击叠加 → 绝对寂静`。

## 06 场景与地点结论

佛国段的空间必须作为一个可连续继承的系统制作：远山寺宇建立宏观环境；殿内建立女主、比丘尼和光柱位置；殿门—庭院—树下讲经区建立唯一空间轴线。主观镜头允许压缩空间，但客观远景必须回到同一轴线。

轮回段场景按匿名化时代设计依据分别锁定建筑语汇，不再使用泛化古装场景。王府深院采用明代宗室院落，灯下内室采用西汉临邛富户语汇，帘后空间采用晚明江南城市内室；其余分别采用西汉宫廷、7世纪末洛阳明堂权力空间、盛唐驿站、东汉边塞、清初盛京政治空间和中晚唐道观。各时代场景仍通过动作匹配和同一身份连接，避免成为艳俗古装混剪。

## 07 场景光影结论

主场景光影的核心不是“神圣光”，而是真实环境光关系：佛国山雾漫射、殿内侧向光柱边缘、庭院树下常态漫射、主观照见时的空间窄化光感、佛前归来的层层剪影光。光线不得主动追随女主，不得产生宗教奇观。

轮回段允许深院低饱和光、灯下短暂柔光、垂帘高位光和火光/风雪叠化，但所有时代色保持低饱和，不能超过青绿落叶的视觉权重。

## 08 道具与特殊现象结论

- 念珠：P0，单珠直径锁定10毫米；灰白哑光石质/菩提质感与深棕檀木均可。传统基数108颗只作内部登记，Prompt不写具体珠数，以珠径、材质、线体柔度、手部接触和逆拨方向维持连续性。
- 青绿落叶：P0，是唯一明确生命色；需锁定叶形、叶脉、湿润感、枝端来源、下落姿态和最终完好状态。
- 发簪：P1，连接“被取代—妒恨”；形制应克制、可跨镜识别，不成为艳俗珠宝。
- 诏书、印章、毒酒杯：P1，服务“冷静报复”的动作接力；文字不可读，印面不使用真实政权标志。
- 主观目光：P0，只是女主心识中的生理真实眼睛；禁止宇宙眼、星河瞳孔、神光和魔眼。
- 心远空间层退：P0，女主尺寸和坐姿不变，殿、庭院、树与远山逐层远离；不表现坠落或超自然爆发。

## 09 群演与规模结论

比丘尼群像是 P0 秩序资产，需在年龄、脸型和微小衣物差异上避免复制，同时保持统一坐姿与拨珠节律。树下听众、宫廷群臣和五百罗汉剪影为 P1 群体配置；五百罗汉不逐一实体化，以少量前景人物、重复纵深、朝向和声音层次建立规模。

## 10 图案、文字与字幕结论

全片唯一允许清晰可读的画内文字是《阿修罗》片名卡。片名字体为近黑深灰底上的灰白枯笔字，不发光、不描金。诏书和印面只保留不可读的形制感，禁止出现真实历史姓名、政权徽记或额外字幕。

## 11 声音资产结论

声音生产默认启用。女主声纹必须跨 VO 与镜头29可见短句保持同一身份；佛国段声音克制、遥远、无表演腔；轮回段通过心跳、衣料、断弦、印章、酒杯、宫门、风雪和喘息加速，而不是持续音乐铺底。

镜头22“一小段弦乐完整奏完”已锁定为画外但属于场内空间的同期琴声，不是配乐；它与西汉临邛琴挑/夜奔的匿名设计依据相互呼应。

声音资产生产方式已调整：女主声纹保留独立P0资产，阶段2提供音色资产Prompt供用户外部生成；佛远讲经声和众声低诵保留登记，但由Seedance镜头Prompt原生生成，不单独生产音频资产，失败时再进入后期替换。

## 13 资产关系与连续性摘要

1. 女主所有 CST 必须继承 `ASR-CHR-P0-001`，不允许通过换脸表达轮回。
2. 念珠从镜头01声音先行，到04、13、18、19承担同一物理道具和同一节律逻辑。
3. 青绿落叶从枝端脱离、慢速下沉、片名影子、佛前接住均为同一叶形和色彩锚点。
4. 主观目光与客观佛必须成对生产：主观像只照见她，客观佛从未转头或选择。
5. 殿内、殿门、庭院和树下讲经区共享固定方向；主观空间变化不能破坏客观轴线。
6. 轮回动作接力固定为：伸手—发簪—握掌—推杯—落印—合拢落叶。
7. 声音演变固定为：秩序—缺拍—中断—呼吸/心跳凸显—人间冲击—绝对寂静。
8. 青绿落叶是唯一明确生命色；压暗红、旧金、蓝灰只能作为低饱和时代色。
